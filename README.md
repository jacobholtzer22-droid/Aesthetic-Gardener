# Aesthetic Gardener Demo Website

## 🌿 Overview

This is a **demonstration website** for Aesthetic Gardener, showcasing a modern, curved design approach for their landscaping business. This demo is meant to show the potential client what their new website could look like with enhanced design elements, smooth curves, and a more contemporary aesthetic.

## ✨ Key Features

### Design Elements
- **Organic Curves**: Smooth, flowing section dividers and rounded corners throughout
- **Blob Animations**: Morphing organic shapes that add visual interest
- **Custom Color Palette**: Earth tones, forest greens, and sage colors that reflect the landscaping industry
- **Premium Typography**: Cormorant Garamond for display text and Outfit for body text
- **Smooth Animations**: Fade-in, slide-up, and stagger animations on scroll
- **Responsive Design**: Fully responsive across all device sizes

### Sections Included
1. **Hero Section**: Eye-catching landing with stats and CTA buttons
2. **Services**: Four main service cards with hover effects
3. **Gallery**: Masonry-style project showcase with hover overlays
4. **About**: Company story with values and testimonials
5. **Contact**: Contact form and business information
6. **Footer**: Comprehensive footer with links and social media

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🎨 Design Philosophy

### Modern Curves & Organic Design
- Replaced straight edges with flowing curves
- Added animated blob shapes for visual interest
- Implemented smooth section transitions with SVG curves
- Used rounded corners (2rem, 3rem) extensively

### Color Scheme
- **Forest Green** (#2b764b - #60b080): Primary brand color
- **Sage Green** (#7a8b61): Secondary accent
- **Earth Tones** (#9d7f61): Warmth and natural feel
- **Light backgrounds**: Soft cream (#fafaf8) and gentle gradients

### Typography
- **Display Font**: Cormorant Garamond (serif) - Elegant and professional
- **Body Font**: Outfit (sans-serif) - Clean and modern
